Phobos 1.0.1
17-Nov-2003
---------------------------------------------------------------
Gestionnaire de Post-it virtuels
Ce logiciel est sous licence VRAF version 462
---------------------------------------------------------------

Cliquez sur l'exécutable pour lancer le programme. Une 
icone apparait alors dans la barre des tâches, à côté de l'horloge. 
Un clique-droit fait apparaitre un menu contextuel, vous 
pouvez ajouter de nouveaux Post-it ou faire afficher ceux 
existants en les sélectionnant.

Dans un Post-it, un clique-droit fait apparaitre un menu avec 
lequel vous pouvez gérer le Post-it (sauvegarde, destruction, 
etc. )

Note : tous les Post-it sont sauvegardés à la fermeture du 
programme (quand on clique sur "Quitter" dans le menu 
principal).

Conseil : mettre un raccourcis vers Phobos.exe dans le menu 
démarrer->démarrage pour démarrer le programme en même temps 
que Windows.
