


Format du fichier XML
-----------------------

Chaque livre est enregistré dans une balise <book> qui contient les balises suivantes :

<titre original=""></titre>		: titre du livre
attribut : titre original


<auteur></auteur>	: auteur


<type></type>

différents types :

1 : roman
2 : reccueil de nouvelles

<genre></genre>

différents genres :

0 : Divers (défaut)
1 : Science-Fiction
2 : Fantastique
3 : Policier
4 : Espionnage
5 : Fiction
6 : Biographie
7 : Roman historique
8 : Roman d'aventure/action
9 : Roman d'amour
10 : Roman de guerre
11 : Poésie
12 : Théâtre
